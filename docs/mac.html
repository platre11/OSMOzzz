macOS 12+

macOS Setup

Get OSMOzzz running on your Mac in under 5 minutes.

Step 1

Download

Click the button below to download the latest release.

Download osmozzz-macos.zip

Once downloaded, unzip the file. You'll get a folder with two files:

# Contents of osmozzz-macos.zip
osmozzz
libonnxruntime.dylib
Step 2

Install

Open Terminal, navigate to the unzipped folder and run:

# Go to the unzipped folder
cd ~/Downloads/osmozzz-macos

# Make it executable
chmod +x osmozzz

# Install (copies binary + registers as MCP server)
./osmozzz install
macOS Gatekeeper warning? If macOS says "cannot be opened because it is from an unidentified developer", go to System Settings → Privacy & Security and click "Allow Anyway".
Step 3

Grant Full Disk Access

OSMOzzz needs Full Disk Access to read your Chrome history, iMessage, Safari, Notes and Calendar.

1 Open System Settings → Privacy & Security → Full Disk Access
2 Click the + button
3 Navigate to ~/.cargo/bin/ and select osmozzz
4 Make sure the toggle is ON
Without Full Disk Access, Chrome, Safari, iMessage, Notes and Calendar will not be indexed.
Step 4

Start the daemon

The daemon indexes your data in the background and keeps it up to date.

osmozzz daemon

The daemon starts automatically on login (installed by osmozzz install). You can also open the local dashboard at:

http://localhost:7523

The dashboard lets you see indexed sources, search your data, and configure cloud connectors.

Step 5

Connect Claude Desktop

osmozzz install automatically registers OSMOzzz as an MCP server in Claude Desktop. Just restart Claude Desktop and it's done.

To verify, open Claude Desktop → Settings → Developer → you should see osmozzz listed as an MCP server.

Don't have Claude Desktop yet? Download it here →
Step 6

Verify it works

Open Claude Desktop and ask:

"What are my most recent emails?"

Or check how many documents are indexed:

osmozzz status
✓ OSMOzzz is ready. Your local sources (Chrome, Safari, iMessage, Notes, Calendar, Terminal, Files) are being indexed automatically.
