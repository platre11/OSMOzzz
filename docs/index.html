Built different.

No cloud sync. No data broker. Just your data, on your machine.

🔒

100% local

Your data never leaves your machine. OSMOzzz runs entirely on your Mac — no external API, no cloud database.

⚡

Semantic search

Powered by ONNX + LanceDB. Search across all your sources at once with natural language queries.

🔌

MCP native

Works natively with Claude Desktop via the MCP protocol. 24 tools available to Claude out of the box.

🌐

P2P network

Share your indexed data with teammates over a secure peer-to-peer network. Data stays on each machine.

📄

PDF search

Search inside PDF content instantly — no indexing required. Reads directly from disk on every query.

🛡️

Privacy filters

Automatic redaction of credit cards, IBANs, API keys and personal data before any result is returned.
