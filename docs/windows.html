Windows Setup

3 steps. No account. No terminal. No subscription.

Note: On Windows, macOS-specific sources (iMessage, Safari, Apple Notes, Apple Calendar) are not available. Chrome, Files and cloud connectors work normally.
Step 1

Download

Click the button to download OSMOzzz for Windows. The file is about 30 MB.

Download for Windows

Once downloaded, right-click the zip file and select "Extract All". You'll see a folder called osmozzz-windows.

Step 2

Open

Open the osmozzz-windows folder and double-click on osmozzz.exe to launch it.

Windows security warning? If Windows shows a "Windows protected your PC" message, click "More info" then "Run anyway". This is normal for open-source apps not distributed through the Microsoft Store. OSMOzzz is fully open source — you can verify every line of code.

OSMOzzz will launch in the background and automatically open the dashboard in your browser.

That's it — open the dashboard

OSMOzzz is running. Open your browser and go to:

localhost:7878

From the dashboard you can:

✓ See all your indexed sources and how many documents are indexed
✓ Search across all your data with natural language
✓ Connect cloud sources (Notion, GitHub, Slack…) by adding your token
✓ OSMOzzz is indexing your local data automatically. Chrome history, Files (Desktop, Documents, Downloads) and Terminal history are picked up with no extra configuration.

What gets indexed automatically

SourceStatus
🌐 Chrome history ✓ Auto
📁 Files (Desktop, Documents, Downloads) ✓ Auto
💻 PowerShell / Terminal history ✓ Auto
📧 Gmail ✓ Via config
◻ Notion, ⌥ GitHub, # Slack… ✓ Via config
💬 iMessage macOS only
🧭 Safari macOS only
📝 Apple Notes macOS only
📅 Apple Calendar macOS only

Cloud sources (Notion, GitHub, Slack, Jira…) can be added from the Configuration tab in the dashboard by adding your API token.
